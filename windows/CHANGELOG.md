# 更新日志

## v0.1.1 (2026-05-19)

### 新增
- 模块设置页面支持自定义样式 (`settings.css`)
- 字段定义支持 `width` 属性设置输入框宽度
- 推送内容显示变化的数据详情

### 优化
- 点检模块推送内容简化，只显示门店名称和月度数据
- 模块配置独立化，每个模块管理自己的门店列表
- 数据与模块分离存储（`data/` 目录）
- 进入设置页面时正确加载已保存的定时配置
- ADB 端口改为全局配置，模块不再单独配置
- **点检数据保存逻辑优化**：用年度数据判断变化，避免月度清空时保存无效数据
- **数据有效性检查**：月度和年度都为0时跳过保存

### 修复
- 修复定时配置加载后显示默认值的问题
- 修复调度器语法错误导致模块无法加载的问题
- 修复前端日志不刷新的问题
- 移除不需要的超时控制代码

### 改进
- 点检模块添加详细日志输出，便于诊断问题
- 模块目录结构优化

## v0.1.0 (2026-05-14)

### 初始版本
- 模块化架构：每个功能模块独立管理
- 可视化调度：任务等待中 → 排队中 → 运行中
- 灵活定时：支持每天、每周、间隔、自定义 cron
- 通知推送：支持企业微信 Webhook 和邮件通知
- 实时日志：前端实时显示任务执行日志
